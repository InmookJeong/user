# 사용자 모듈(프로젝트)

## 모듈(프로젝트) 명칭
- user

## 기술 스팩 및 개발 환경
- JDK : Oracle JDK 1.8
- 백엔드 프레임워크 : SpringFramework (version : 4.x)
- 서버 : Apache Tomcat (version : 9.x)
- 데이터베이스 : MySQL (version : 8.x)
- API 문서화 도구 : Spring REST Docs
- API 테스트 도구 : Postman
- 빌드 도구 : Maven
- 개발 도구 : Eclipse
